LATEX

Что такое LaTeX?

LaTeX — наиболее популярный набор макрорасширений (или макропакет) системы компьютерной вёрстки TeX, который облегчает набор сложных документов. В типографском наборе системы TeX форматируется традиционно как LAΤΕΧ.

Что такое TeX?

TeX — система компьютерной вёрстки, разработанная американским профессором информатики Дональдом Кнутом в целях создания компьютерной типографии. В неё входят средства для секционирования документов, для работы с перекрёстными ссылками. В частности, благодаря этим возможностям, TeX популярен в академических кругах, особенно среди математиков и физиков.

Дистрибутивы TeX

Если вы новичок в TeX и LaTeX или просто хотите простую установку, получите полный дистрибутив TeX. TeX Users Group (TUG) имеет список известных дистрибутивов, которые являются полностью или наименее в первую очередь свободным программным обеспечением.

  • Linux

    Проверьте свой источник программного обеспечения дистрибутивов Linux на наличие дистрибутива TeX, включая LaTeX. Вы также можете установить текущий дистрибутив TeX Live напрямую - на самом деле это может быть целесообразно, так как многие дистрибутивы Linux содержат только старые версии TeX Live, подробности см. в статусе пакета Linux TeX Live.

  • Apple

    Дистрибутив MacTeX содержит все необходимое, включая полную систему TeX с самим LaTeX и редакторы для написания документов.

  • Windows

    Ознакомьтесь с дистрибутивами MiKTeX orproTeXt или TeX Live; они содержат полную систему TeX с самим LaTeX и редакторами для написания документов.

  • Online

    Онлайн-сервисы LaTeX, такие как Papeeria, Overleaf, ShareLaTeX, Datazar и база LaTeX, предлагают возможность редактировать, просматривать и загружать файлы LaTeX и полученные PDF-файлы.
