Создание первого TeX файла

Чтобы начать работать в LaTeX Вам нужно либо создать файл с расширением .tex, либо воспользоваться онлайн редактором, например, Overleaf.

Первым делом необходимо через ключевое слово \documentclass объявить класс документа, у разных документов свои классы, например, класс для резюме не будет совпадать с классом для статей. В нашем случае используем самый простой класс LaTeX article.

\documentclass{ article }

Далее необходимо объявить, что чтение документа началось, используем команды \begin{ document }, \end{ document }. Объединим все что написал, добавим базовый текст и запустим программу.

\documentclass{ article }
\begin{ document }
Hello, World!
\end{ document }

Теперь, когда документ уже создан, можно задуматься о его внешнем виде, поговорим о том, что необходимо подключить в первую очередь. Это называется преамбулой документа.

\documentclass[12pt, letterpaper]{ article }

Как уже говорилось, этой командой определяется тип документа. В нее также можно передавать дополнительные параметры, включенные в квадратные скобки через запятую. В данном примере эти параметры устанавливают размер шрифта (12pt) и размер страницы (letterpaper). Конечно же, для шрифта можно установить и другие размеры (9pt, 11pt, 12pt), но если размер не задан, то предустановленным значением будет 10pt. Что же касается размера страницы, то для него доступны варианты a4paperи legalpaper.

\usepackage[utf8]{ inputenc }

Эта команда задает кодировку документа. Ее можно опустить либо изменить на другой вариант, но рекомендуется использовать именно utf-8. Если вам не требуется конкретно другая кодировка, либо вы просто не уверены, то добавьте эту строку во вступление.

Подключим русский язык, потому что изначально LaTeX был рассчитан для латиницы

\usepackage[russian]{ babel }

Знак % объявляет начало комментария
Для добавления в документ заголовка, автора и даты необходимо внести во вступление три строки (только не в основное тело документа):

\title{ Первая страничка } % заголовок
\author{ Александр Лотяков } % имя автора
\date{ Ноябрь 2022 } % дата

Теперь добавим отступы.

\usepackage[ left=1.5cm, right=1.5cm, top=2cm,
bottom=2.5cm ]{ geometry }

И призовем нашу информацию об авторе внутри документа с помощью команды \maketitle. Теперь объединим все в одно целое и запустим.

\documentclass{ article }
\documentclass[12pt, letterpaper]{ article }
\usepackage[utf8]{ inputenc }
\usepackage[russian]{ babel }
\usepackage[ left=1.5cm, right=1.5cm, top=2cm,
bottom=2.5cm]{ geometry }
\title{ Первая страничка }
\author{ Александр Лотяков }
\date{ Ноябрь 2022 }
\begin{ document }
\maketitle
Пивет, я родився!
\end{ document }
