Форматирование текста

Текст является важной частью любой книги, статьи, языка программирования и даже LaTeX не исключение, в этом разделе мы рассмотрим основные способы форматирования текста

\underline{ Подчеркнутый }
Подчеркнутый
\textbf{ Полужирный }
$$\textbf{Полужирный}$$
\textit{ Курсив }, \em{ Курсив }
$$\textit{Курсив}$$

Вставка кода

Чтобы код не потерял свой внешний вид в LaTeX было создано множество пакетов, рассмотрим несколько из них. Для демонстрации результата воспользуемся онлайн-сервисом Overleaf.

Пакет Verbatim

Данный пакет максимально прост в использовании, вам лишь нужно прописать \begin{verbatim}, \end{verbatim} и вставить нужный код

\begin{verbatim}
for i in range(10):
  print("Hello, world!)
\end{verbatim}

Пакет Alltt

Данный пакет дает больше возможностей, чем пакет verbatim, вы сможете менять форматирование текста и даже выделять нужный текст цветами

\begin{alltt}
for i in range(10):
  \textit{print}("Hello, world!)
\end{alltt}

Пакет Minted

Данный пакет подходит для тех, кто хочет не просто передать сам код, но и синтаксическую подсветку языка, на котором пишет. Ключевой момент в том, чтобы после объявления начала блока кода написать название языка, на котором код написан.

\begin{minted}{python}
for i in range(10):
  print("Hello, world")
\end{minted}
